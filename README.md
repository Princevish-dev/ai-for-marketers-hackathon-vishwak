# 🚀 GROWTHOS: The Ultimate AI Marketing Suite

**Built for the AI for Marketers Hackathon** 🏆

GrowthOS is an end-to-end, execution-driven marketing automation platform. It transforms how businesses generate content, optimize ads, and analyze competitors—all powered by **Google Gemini AI**. GrowthOS focuses on **real business outcomes, automated workflows, and measurable ROI**.

## 🔥 Why GrowthOS? (Hackathon Alignment)
- **AI Content Engine:** Repurpose a single blog post into dozens of viral Twitter, LinkedIn, Instagram, and Newsletter assets instantly.
- **AI Ads Optimization:** Automatically generate high-converting Google, Facebook, and Instagram ad copy alongside a **Live HTML Landing Page**.
- **Customer Insights:** Scrape competitor weaknesses and sentiment, then feed those insights directly into your content generation workflow.
- **ROI & Analytics Dashboard:** A unified dashboard that calculates exact hours and money saved (ROI) for your team, proving measurable impact.

---

## ⚙️ Features & Interconnected Workflows

### 1. ✍️ Content Repurposer
Turn long-form text or blog URLs into a week's worth of multi-channel social media posts formatted specifically for each platform's algorithm.

### 2. 🎯 Ad Studio
Generate targeted ad copies. *Magic Feature:* Click **"Push to Ad Studio"** from the Content Repurposer to automatically pre-fill your ad context. The Ad Studio also codes a **beautiful, mobile-responsive HTML Landing Page** on the fly.

### 3. 🕵️ Competitor Intel
Extract complaints, positioning, and sentiment for any brand. *Magic Feature:* Click **"Use Insights"** to automatically send competitor weaknesses to the Content Repurposer so your AI writes content specifically designed to beat them.

### 4. 📊 ROI Dashboard
Watch your savings grow. The dashboard calculates the exact INR/USD saved based on your team size and average salary, proving the financial value of the AI suite.

## 🚀 How to Run Locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/HackIndiaXYZ/ai-for-marketers-hackathon-vishwak.git
   cd ai-for-marketers-hackathon-vishwak
   ```

2. **Install Dependencies**
   ```bash
   pip install -r requirements.txt
   ```

3. **Set your API Key**
   Create a `.env` file in the root directory and add your Gemini API key:
   ```env
   GEMINI_API_KEY="AIzaSy..."
   ```
   *(Alternatively, you can securely enter it via the app's sidebar).*

4. **Launch the App**
   ```bash
   streamlit run app.py
   ```
